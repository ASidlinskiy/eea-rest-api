== Description ==


An Event Espresso 4 plugin for providing an RESTful interface for Event Espresso's events data.

Documentation, installation and usages guides are located on developer.eventespresso.com/ee-plugin/ee4-json-rest-api/

This plugin also requires you have the master branch of the WP API active on your site.

== Installation ==

1. After unzipping, upload everything in the `event-espresso` folder to your `/wp-content/plugins/` directory (preserving directory structure).

2. Activate the plugin through the 'Plugins' menu in WordPress.
